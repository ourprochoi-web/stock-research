## 3. 프로토콜 & 인프라

### C1. MCP (Model Context Protocol)

| 항목 | 수치 | 시점 |
|------|------|------|
| 런칭 | 2024.11.25 | Anthropic |
| 공개 서버 수 | 10,000+ | 2025.12 |
| 월간 SDK 다운로드 | 97M+ (Python + TypeScript) | 2025.12 |
| GitHub 리포지토리 (mcp-server 토픽) | 15,926 | 2026.05.24 |
| 프로덕션 도입 기업 | 41% (조사 대상 SW 조직) | 2026, Stacklok |

- **거버넌스 전환**: 2025.12 Anthropic이 MCP를 Linux Foundation 산하 Agentic AI Foundation에 기부. Anthropic, Block, OpenAI가 공동 설립. HTTP/TCP/IP와 같은 수준의 공유 인프라로 포지셔닝.
- **플랫폼 지원**: ChatGPT, Cursor, Gemini, Microsoft Copilot, VS Code 등 주요 AI 플랫폼/IDE가 MCP 클라이언트·서버·커넥터를 지원.
- **의미**: AI 에이전트가 외부 도구·데이터·서비스에 접근하는 "USB-C" 표준. SaaS 기업들이 MCP 서버를 제공하지 않으면 AI 에이전트 생태계에서 소외될 위험.

### C2. A2A (Agent-to-Agent Protocol)

| 항목 | 수치 | 시점 |
|------|------|------|
| 런칭 | 2025년 (Google 주도) | Google |
| 참여 조직 | 150+ | 2026.04 |
| GitHub 스타 | 22,000+ | 2026.04 |
| 프로덕션 SDK 언어 | 5개 (Python, JS, Java, Go, .NET) | 2026.04 |

- **MCP vs A2A**: MCP는 에이전트↔도구 간 연결(도구 호출), A2A는 에이전트↔에이전트 간 통신(업무 위임·협업). 상호 보완적 관계.
- Linux Foundation 산하에서 관리. Google, Microsoft, AWS 플랫폼에 딥 통합.
- **적용 수직**: 공급망, 금융서비스, 보험, IT 운영 등에서 교차 벤더·교차 환경 자율 시스템 조율에 활용.

### D3. SaaS 대체 논쟁 (SaaSpocalypse)

- **용어 등장**: 2026년 2월, 전통 SaaS 기업 밸류에이션의 급격한 하락을 지칭.
- **트리거**: 2026년 1월 Anthropic의 Claude Cowork·Claude Code 출시 → AI 에이전트가 기존 SaaS 워크플로우를 대체할 수 있다는 인식 확산.
- **시장 충격**: 48시간 내 SaaS 기업 시가총액 $285B 증발. 30일간 $2T 시총 소실.
- **최대 피해**: Atlassian(-35%), Salesforce(-28%) — 태스크 트래킹·데이터 입력·고객 로깅 등 AI 에이전트 자동화에 가장 취약한 워크플로우.
- **핵심 위협**: per-seat SaaS 과금 모델의 존재론적 위기. AI 에이전트가 태스크 수준 업무를 처리하면 human seat 수요 감소.
