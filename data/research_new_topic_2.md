# 로봇/휴머노이드 리서치 시리즈 -- 로드맵 및 초기 리서치

> 목적: 산업용 로봇 -> 협동로봇(Cobot) -> 휴머노이드로 진화하는 로봇 산업의 밸류체인을 분석. "자동화율-단가 하락-범용성 확대" 프레임으로 한국/글로벌 종목의 투자 좌표를 찍음. Physical AI 편과의 시너지 극대화.
> 최종 갱신: 2026-06-23
> 상태: **로드맵 수립 + 초기 리서치 완료**

## 2. 시리즈 전체 구성 (8편)

### Part 1 -- 로봇 산업의 대전환: 왜 지금 휴머노이드인가
- 산업용 -> 협동 -> 자율 -> 휴머노이드 진화사
- 매크로 수요 전망 (IFR, Goldman Sachs, Morgan Stanley)
- 4단계 밸류체인 맵 (부품 -> 하드웨어 -> 소프트웨어/AI -> 서비스/통합)
- "자동화율-단가-범용성" 투자 프레임워크
- 악센트: RobotCyan `#06B6D4`

### Part 2 -- 로봇주 투자 지도: 종목 크로스 밸류에이션
- 20개+ 종목 크로스 밸류에이션, 글로벌 멀티플 비교

- 리스크 체크리스트, 포트폴리오 구성

## 5. 핵심 투자 테마 5개

### 테마 1: 휴머노이드 상용화 카운트다운 -- "2028년이 X-Day"
- **핵심**: Tesla/현대/삼성 모두 2028 상용화 목표. 2026은 프로토타입 -> 파일럿 전환기
- **데이터**: Tesla Fremont 100만대 목표, 현대 Atlas 30,000대(2030), 삼성-레인보우 2028
- **수혜 기업**: Tesla, 현대차, 레인보우로보틱스, Figure AI
- **리스크**: 상용화 지연 히스토리 (로봇 산업의 고질적 문제), 소프트웨어 완성도

### 테마 2: K-휴머노이드 얼라이언스 -- "정부 7,700억 + 재벌 투자"
- **핵심**: 한국 정부 주도 K-휴머노이드 생태계 구축, 삼성/현대/두산 3강 체제
- **데이터**: 정부 7,700억원, 삼성 레인보우 지분 35%, 현대 Boston Dynamics 100% 인수
- **수혜 기업**: 레인보우로보틱스, 두산로보틱스, 부품 서플라이체인
- **리스크**: 미국/중국 대비 기술 격차, 내수 시장 규모 한계

### 테마 3: 협동로봇 -- "CAGR 27.5%의 확실한 성장"
- **핵심**: 휴머노이드는 미래, 코봇은 현재. 제조업 자동화율 상승의 직접 수혜
- **데이터**: 코봇 시장 $1.3B(2024) -> $7B+(2030), CAGR 27.5%
- **수혜 기업**: 두산로보틱스, Universal Robots(Teradyne), Fanuc, 뉴로메카
- **리스크**: 중국 저가 코봇 공세, 한국 기업 글로벌 점유율 아직 미미

### 테마 4: 로봇의 두뇌 -- "Foundation Model이 게임 체인저"
- **핵심**: NVIDIA Isaac + Google RT-X + OpenAI 로봇 파운데이션 모델이 범용 로봇 실현 가속
- **데이터**: NVIDIA Isaac Sim 사용자 급증, Figure+OpenAI 대화형 로봇 데모
- **수혜 기업**: NVIDIA, Google DeepMind, Tesla (FSD 전이), 삼성(자체 AI)
- **리스크**: Sim-to-Real 격차, 범용 모델의 안전성/신뢰성

### 테마 5: 부품 서플라이체인 -- "숨겨진 삽 파는 기업"
- **핵심**: 감속기/액추에이터/센서는 로봇 BOM의 핵심. 일본 독점 구조에 한국 기업 진입
- **데이터**: Harmonic Drive 글로벌 점유율 ~60%, 휴머노이드 1대당 40-50개 액추에이터
- **수혜 기업**: 삼익THK, SFA, 로보스타, Harmonic Drive, Nabtesco
- **리스크**: 일본 부품 의존도, 내재화 속도

## 6. 밸류체인 맵

```
[로봇/휴머노이드 밸류체인]

1. 핵심 부품 (Components)
   ├── 감속기 (Reducer)             → Harmonic Drive, Nabtesco, 삼익THK
   ├── 모터/액추에이터              → Maxon, TQ Robotics, 삼성전기
   ├── 센서 (Force/Torque/LiDAR)   → ATI, OnRobot, 오로라
   ├── 배터리/전원                  → Samsung SDI, LG엔솔, CATL
   └── 반도체/컴퓨트                → NVIDIA (Jetson/Thor), Qualcomm

2. 하드웨어 플랫폼 (Hardware)
   ├── 휴머노이드                   → Tesla, Figure AI, 레인보우로보틱스
   ├── 협동로봇 (Cobot)            → 두산로보틱스, Universal Robots, Fanuc
   ├── 산업용 로봇 (Arm)           → Fanuc, ABB, KUKA
   ├── 이동로봇 (AMR/AGV)          → 현대로보틱스, Locus, Symbotic
   └── 의료 로봇                   → Intuitive Surgical, Stryker

3. 소프트웨어/AI (Intelligence)
   ├── 로봇 OS/미들웨어            → ROS2, NVIDIA Isaac
   ├── Foundation Model             → Google RT-X, OpenAI, Tesla FSD 전이
   ├── Sim-to-Real 시뮬레이션       → NVIDIA Omniverse, Unity
   └── Fleet Management            → Brain Corp, AWS RoboMaker

4. 시스템 통합/서비스 (Integration)
   ├── 공장 자동화 SI              → SFA, 현대위아, Rockwell
   ├── 물류 자동화                  → Symbotic, Amazon Robotics
   ├── RaaS (Robot-as-a-Service)   → Formic, Bear Robotics
   └── 유지보수/MRO                → 각 OEM 직접 + 3rd party
```
