# Claude App 프롬프트 — AI 전력 인프라 리서치

아래 프롬프트를 Claude App에 붙여넣어서 사용하세요.

---

당신은 산업 리서치 애널리스트입니다. 아래 주제에 대해 한국어로 된 단일 HTML 리서치 문서를 작성해주세요.

## 주제
**AI 전력 인프라 투자 지도 — 지능을 켜려면 전기가 필요하다**

AI 데이터센터의 폭발적 전력 수요 증가를 기점으로, 전력 공급 밸류체인 전체를 분석하고 투자 기회를 매핑하는 리서치입니다.

## 다뤄야 할 구조

### 1. 매크로 — AI가 만드는 전력 수요의 규모
- 글로벌 AI 데이터센터 전력 소비 전망 (2024→2028)
- 단일 GPU 클러스터의 전력 소비 규모 (MW 단위)
- "전력이 AI의 진짜 병목"이라는 테제 설정

### 2. 밸류체인 맵 — 전력은 어디서 오는가
전력 생산 → 송배전 → 데이터센터 전력 인프라 → 냉각 순서로 체인을 분해

**발전 (전력 생산)**
- 원전/SMR: 뉴스케일, 한수원, 두산에너빌리티
- 가스터빈/LNG: GE버노바, 두산에너빌리티, 한화파워시스템
- 재생에너지: 태양광·풍력의 한계와 보완 역할

**송배전 (전력 운반)**
- 변압기·전력케이블: HD현대일렉트릭, LS일렉트릭, 대한전선
- 글로벌 변압기 부족 이슈와 리드타임

**데이터센터 전력 인프라**
- UPS·배전반·전력반도체
- 직류(DC) 배전 트렌드

**냉각 (열 관리)**
- 공냉 → 수냉 → 액침냉각 전환
- 관련 종목: 삼화콘덴서, 고영테크놀로지 등

### 3. 종목 분석 — 핵심 플레이어별 투자 포인트
각 종목마다:
- 사업 구조와 AI 전력 연결 고리
- 핵심 숫자 (매출, 수주잔고, 밸류에이션)
- Bull case / Bear case
- 모니터링 지표 (킬 크라이테리아)

### 4. 투자 관점 정리
- 단기(6개월) vs 중기(2년) 유망 구간
- 리스크 요인 (정책, 경쟁, 기술 전환)

## 디자인/포맷 요구사항

1. **단일 HTML 파일**로 완성 (외부 JS/CSS 의존 최소화, Pretendard 폰트 CDN만 사용)
2. **다크 테마** 기반 — 배경은 어두운 네이비/차콜 계열
3. **반응형**: 모바일에서도 읽기 편해야 함
4. **sticky 네비게이션**: 섹션 이동 가능한 상단 고정 메뉴
5. **종목 카드**: 각 종목을 카드 UI로 정리 (뱃지, 핵심 수치, bull/bear, 킬 크라이테리아)
6. **밸류체인 다이어그램**: 발전→송배전→DC인프라→냉각 흐름을 CSS로 시각화
7. 한국어 사용, 전문적이되 읽기 쉬운 톤

## 애니메이션 (매우 중요 — 화려하게)

히어로와 전체 페이지에 걸쳐 시각적 임팩트가 강한 애니메이션을 적용해주세요. 아래 효과를 모두 포함:

### 히어로 섹션 (풀스크린)
- **배경**: 다중 레이어 오로라 그라디언트가 느리게 이동/변형 (3개 이상의 radial-gradient, CSS animation으로 위치·크기·투명도 순환)
- **파티클**: JS로 생성하는 50개 이상의 빛나는 입자. 다양한 크기(2~6px), 4가지 이상 색상, glow(box-shadow), 각각 다른 속도·딜레이로 부유. 일부는 위로, 일부는 아래로
- **그리드 라인**: 은은한 격자 배경 + radial mask로 중심부만 보이게
- **제목**: gradient text (3색 이상) + background-size 300%로 색상이 흐르는 shimmer 애니메이션
- **텍스트 등장**: eyebrow → 제목 → 부제 → 통계 순으로 staggered fade-up (각 0.2~0.3초 간격)
- **통계 숫자**: 0에서 목표치까지 카운트업 애니메이션
- **스크롤 유도**: 하단에 펄스 애니메이션이 있는 scroll indicator

### 밸류체인 다이어그램
- 각 단계가 스크롤 시 순차적으로 나타나며 (staggered reveal)
- 단계 간 연결선에 에너지가 흐르는 듯한 CSS 애니메이션 (dashed border animation 또는 gradient flow)
- 각 노드에 hover 시 glow 효과

### 종목 카드
- IntersectionObserver로 viewport 진입 시 fade-up + slight scale
- 카드 간 staggered delay (0.1초 간격)
- hover 시 border glow + 미세한 lift + shadow 확대
- Bull/Bear 섹션에 색상 코딩 (초록/빨강 계열 glow)

### 섹션 전환
- 각 섹션 헤더에 typing 느낌의 등장 효과 또는 글자 단위 staggered fade-in
- 섹션 간 subtle한 구분선 애니메이션 (가로선이 좌→우로 그려지는 효과)

### 전체 공통
- CSS transition과 animation 위주로 구현 (JS는 파티클 생성과 IntersectionObserver에만 사용)
- `prefers-reduced-motion` 미디어 쿼리 존중
- 60fps 유지를 위해 transform/opacity 위주로 애니메이팅

최신 데이터를 웹 검색해서 반영해주세요. 특히 2025-2026년 기준 전력 수요 전망, 주요 종목의 최근 실적/수주 데이터가 중요합니다.
