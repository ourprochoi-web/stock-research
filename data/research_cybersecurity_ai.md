- Gartner, 2025년 6월 "AI SOC Agents"를 공식 카테고리로 지정
- 2026년 말까지 대기업 SOC 워크플로의 **30%+** 가 에이전트 AI로 실행 예상
- 자율 알림 분류 → 분석가 워크로드 **80%+** 감소
- AI/자동화 고도입 기업: 침해 건당 **$1.9M** 절감, 침해 생애주기 **80일** 단축
- 일평균 알림 **10,000건+**, 오탐률 50%+, 미조사 알림 40% — AI SOC의 핵심 해결 과제
- 주요 플랫폼: CrowdStrike Charlotte AI, Palo Alto XSIAM, SentinelOne Purple AI, Stellar Cyber, Exaforce

### 2.4 LLM 보안 위협

| 위협 유형 | 설명 |
|----------|------|
| 프롬프트 인젝션 (OWASP LLM01:2025) | 시스템 프롬프트와 사용자 입력 분리 불가능 악용. #1 취약점 |
| 탈옥(Jailbreaking) | 안전 프로토콜 무시하는 입력 패턴 |
| 간접 인젝션 | 이메일·문서·웹사이트·이미지에 숨겨진 악성 지시 |
| 모델 포이즈닝 | 학습 데이터(Common Crawl, Reddit, GitHub)에 백도어 매립 |
| 데이터 유출 | RAG·에이전트 시스템을 통한 민감정보 추출 |
| 도구 포이즈닝 (MCP) | Model Context Protocol 확장으로 에이전트 공격 표면 급확대 |

- AI 에이전트·MCP 확산 → 공격 표면이 "모델" 에서 "도구 체인"으로 확장
- 방어 패러다임: "예방 전용" → **"침해 가정(assume breach)"** 심층 방어 전환
