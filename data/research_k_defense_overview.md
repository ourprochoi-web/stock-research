## 1. 글로벌 방산 시장 구조

### 1-1. 세계 무기수출 점유율 (SIPRI, 2021~2025 누계)

| 국가 | 점유율 | 비고 |
|------|--------|------|
| 미국 | 42% | 절대강자 · 고급 노드 독점 + 승인권(ITAR) |
| 프랑스 | ~10% | 풀스펙트럼 강국 |
| 러시아 | 6.8% | 풀스펙트럼 (2012년 30% → 2025년 6%로 추락) |
| 독일 · 중국 | 각 ~5% | 풀스펙트럼 강국 |
| 한국 | ~3% | 신흥 특화주자 · 고속 상승 ↑ |

- **상위 5개국**이 전체 수출의 **70%**를 차지하는 초집중 시장
- 미국의 진짜 힘: 점유율 42%가 아니라 **고급 노드의 독점과 승인권**
- 장거리 지대지 미사일(사거리 250km↑) 수출의 **45%가 미국**

### 1-2. 세계 3개 층위 피라미드

| 층위 | 국가 | 특성 |
|------|------|------|
| 절대강자 | 미국 | 글로벌 42%, 엔진·스텔스·우주·미사일 독점, ITAR 승인권 |
| 풀스펙트럼 강국 | 프랑스·러시아·독일·중국·영국·이탈리아 | 전투기·함정·미사일 자체 보유 |
| 신흥 특화주자 | 한국·이스라엘·튀르키예 | 특정 영역에서 빠르게 치고 올라옴 |

### 1-3. 기업 규모 격차 (SIPRI 100대 방산기업, 2024년 매출)

- **록히드마틴** 1곳 방산 매출: **~$60B+** (방산 ~$68B, 전체 매출의 96%)
- **한국 4사 합산**: **$14.1B** (약 141억 달러)
- 록히드마틴 한 곳이 한국 빅4 합산의 약 **4~5배**
- 미국 상위 10개 방산기업 중 **6개가 미국 기업**
- 한국 4사 합산 1년 새 **+31%** 성장, 한화그룹은 **+42%** 증가
- **"규모는 소형, 속도는 최고"**

- 이 5곳은 남에게 의존하는 노드가 사실상 없음

### 1-6. 러시아 — 제재 고립 + 전시경제

- 국영 **로스텍**(100% 국가소유, 칼라시니코프 등 수백 개 기업) + **통합조선공사** 합산 방산매출: **$31.2B** (+23%)
- 세계 수출 점유율: **2012년 30%** → **2025년 6%** 추락
- 전쟁 내수가 매출을 메웠으나, 수출길은 끊김
- 숙련 인력 부족·부품난, 일부 부품을 중국에 의존 시작

## 2. 한국 방산 수출 실적

### 2-1. 주요 수출 수치

| 항목 | 수치 | 비고 |
|------|------|------|
| 2025년 방산 수출 | **$154억** (154억 달러) | 전년比 **+60%** |
| 빅4 합산 매출 (2025) | **40.5조 원** | |
| 유럽 NATO 시장 공급국 순위 | **2위** (8.6%) | 미국(58%) 다음, 이스라엘(7.7%)·프랑스(7.4%) 제침 |
| 5년 수출 중 폴란드 비중 | **~46%** | 좁고 강한 교두보 |
| 수출 대상국 수 | **16개국** | 2022년 7개국에서 확대 |
| 2024년 수출 중 지상·화력 비중 | **73%** (영업이익의 70%) | |
| 빅4 수출 비중 (2024~25) | **~50%** | 체질 전환 중 |
| 한화에어로 수출 비중 | **50%+** (지상방산 71%) | 2025년 처음으로 50% 돌파 |

### 2-2. 한국의 세계 좌표 정리

- 글로벌 5년 누계: 10위권
- 유럽 NATO 시장: **미국 다음 2위 공급국 (8.6%)**
- 특화 영역: 재래식 **지상무기**(자주포·전차·다연장) — 기술 난도 중간, 수요 큰 세그먼트
- 경쟁력: 서방 무기보다 **30~50% 저렴**, 납기 빠름 (남북 대치로 생산라인 수십 년 미중단)
- 천장: 전투기 엔진·스텔스·장거리 정밀타격·대형 잠수함 같은 **고부가 영역에서 하위권**
- 2025년 폴란드 잠수함(오르카) 수주전 **고배** — NATO 결속 논리에 패배

## 3. K-방산 밸류체인

### 3-1. 4계층 구조

| 계층 | 설명 | 특징 |
|------|------|------|
| ① 수요층 | 국내 방사청(독점) ↔ 해외(자유가격) | 손님이 사실상 한 명 |
| ② 체계종합 (Prime) | 빅4 + 함정·전자 | 권력 집중 |
| ③ 협력업체 (1·2차) | 부품·구성품 | 생태계 얇음 |
| ④ 소재·핵심부품 | 엔진·반도체 | 수입의존 병목 |
| (R&D 기반) | ADD 국가주도 | 재원 77.6% 정부, 기업 22.4% |

### 3-2. 도메인별 지도 (체계종합 기업 · 주력 무기)

| 도메인 | 체계종합 | 주력 무기 | 수출 지위 |
|--------|---------|----------|----------|
| 화력·지상 | 한화에어로스페이스 | K9 자주포 · 천무 · 레드백 | 수출 주력 |
| 기동 | 현대로템 | K2 전차 · K808 장갑차 | 폴란드·페루 |
| 유도무기 | LIG넥스원 | 천궁-II · 현궁 · 해성 | 중동 확대 |
| 항공 | KAI | FA-50 · KF-21 · 수리온 | 엔진 족쇄 |
| 함정 | 한화오션 · HD현대중공업 | 호위함 · 잠수함 | 고부가 약세 |
| 전자·레이더 | 한화시스템 | AESA 레이더 · 지휘통제 | 서브시스템 |
| 탄약 | 풍산 | 탄약 · 신관 | 후속군수 |

- 도메인별 사실상 한 곳이 지배 → **국내 경쟁은 없고 수출에서만 글로벌 경쟁**
- 한화그룹: 화력·항공엔진·전자·함정을 동시에 보유 → 가장 **수직통합된 플레이어**

### 3-3. 국산화 매핑 — 강한 노드 vs 병목 노드

**강한 노드 (국산화 높음 = 수출 자유)**

| 영역 | 상세 |
|------|------|
| 화력 — K9 자주포 | 11개국 도입, 사실상 글로벌 표준 |
| 유도무기 | 천궁-II·현궁·해성, LIG넥스원 자체 통제 |
| AESA 레이더 | 2025년 8월 KF-21 전력화, 국산 개발 (한화시스템) |
| 탄약 | 풍산, 후속군수까지 연계 |

**병목 노드 (수입의존 = 수출 천장)**

| 영역 | 상세 |
|------|------|
| 항공엔진 | KF-21=GE F414, FA-50=GE F404 → **ITAR 족쇄** |
| 기동 파워팩 | K2 1차양산 독일 MTU+렝크, 국산화 내구성 이슈 |
| 군용 반도체·센서 | 분산 의존 |
| 특수소재 | 내열합금·복합재 |

- 강한 노드(지상·화력)는 2024년 수출의 **73%**·영업이익의 **70%**를 차지하며 이미 현금화 중
- 약한 노드(항공엔진)는 미국이 쥐고 있어 KF-21 수출을 막음
- **다음 성장의 병목 = 국산화 병목과 정확히 일치**
