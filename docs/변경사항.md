# 리서치 아카이브 — 변경사항 정리

원본 `index.html`(우주·파티클 인트로)을 **검색·인덱싱이 가능한 지식 아카이브**로 재설계했습니다.
결과물: `리서치 아카이브.dc.html` (편집용 원본) / `리서치 아카이브 (standalone).html` (단일 파일, 오프라인 실행용)

---

## 1. 컨셉 전환

| | 원본 | 변경 후 |
|---|---|---|
| 인트로 비주얼 | 우주·파티클·별자리·유성 | **"점을 이어 구조를 드러내는" 지식 네트워크 (3D)** |
| 모션 성격 | 떠다니는 입자 | 부드럽고 유기적으로 흐르는 3D 회전·연결 |
| 핵심 메시지 | 분위기 위주 | *연결 → 구조 → AI의 현실화* 라는 서사 |
| 인트로의 역할 | 장식 | **애니메이션이 먼저 시선을 끌고, 검색은 그 위에** |

핵심 카피도 교체했습니다 → **"점을 잇는 곳에 / 구조가 보인다"**

---

## 2. 시네마틱 오프닝 (구체)

페이지 진입 시 한 번 재생되는 연출을 새로 만들었습니다.

1. **조립 (0 ~ 1.9초)** — 노드들이 가운데에서 바깥으로 퍼지며 구체가 만들어지고, 가까운 노드끼리 선으로 연결됨 → *AI가 구현되는 느낌*
2. **점화 (≈ 2.15초)** — 중심에서 빛의 충격파 링이 퍼지며 밝게 터짐 (완성의 순간)
3. **후퇴 & 등장 (2.45 ~ 4.1초)** — 구체가 살짝 축소되고 옅어지며 뒤로 물러나고, 동시에 **홈페이지(제목·검색바·추천·통계)가 아래에서 떠오름**
4. **정지 상태** — 구체는 은은한 배경(투명도 약 0.36)으로 남아 살아 움직이는 브랜드 요소가 됨

### 구체 자체의 변화 (요청 반영)
- **크게**: 화면을 채우도록 확대 (최대 680px)
- **연결**: 각 노드를 가장 가까운 이웃들과 실제 선으로 연결, 앞쪽일수록 선이 굵고 밝게(깊이감)
- **다이내믹 컬러**: 색상(hue)이 계속 회전하며 보라·핑크·시안·그린·옐로우가 흐르듯 변함
- 캔버스(Canvas) + 3D 투영으로 다시 구현해 부드러운 회전과 원근감 확보

## 3. 인트로 시안 3종 비교

상단 스위처로 배경 애니메이션을 바꿔가며 비교할 수 있습니다.

- **네트워크** — 뒤로 기울어진 3D 평면 위에 노드가 그려지며 연결 (가장 입체적인 깊이)
- **구체** — 회전하는 3D 글로브 *(기본 선택)*
- **레이어** — 화면을 가득 채운 다층 깊이 네트워크 지도

마우스를 움직이면 시차(parallax)로 배경이 부드럽게 기울어집니다.

---

## 4. 검색 · 인덱싱 시스템 (신규)

원본에 없던 핵심 기능을 추가했습니다. 리서치가 늘어나도(수십~수백 개) 찾을 수 있게 설계했습니다.

- **히어로 중앙 검색바** + 추천 키워드 칩(전력 인프라·GPU·데이터센터·로보틱스·자율주행·밸류체인)
- **스크롤 시 나타나는 상단 고정 검색바** — 로고·검색·카테고리 필터·결과 수가 항상 따라옴
- **실시간 필터링** — 입력하는 즉시 결과가 좁혀짐. 검색 대상: 제목 · 설명 · 유형 · 시리즈 · 카테고리 · **태그**
- **카테고리 인덱싱** — 전체 / AI 인프라 / Physical AI 로 즉시 필터
- **태그 부여** — 각 리서치에 키워드 태그를 달아 검색·분류 기준으로 활용
- **결과 없음 상태** — "일치하는 리서치가 없습니다" + 초기화 버튼
- Enter 또는 검색 버튼 → 결과 영역으로 부드럽게 스크롤 이동

---

## 5. 아카이브(목록) 영역

- 카테고리별 섹션 (AI Infrastructure / Physical AI) — 색상 라벨로 구분
- 카드: 유형 배지 · 시리즈 표시 · 제목 · 설명 · **태그** · 날짜
- 호버 시 떠오르는 인터랙션, 클릭하면 실제 리서치 파일로 이동
- 히어로 통계(리서치 수 · 산업 테마 수)는 등장 후 숫자가 카운트업

---

## 6. 기술 / 구조 메모

- 단일 **Design Component**(`.dc.html`)로 구성 — 브라우저에서 바로 열림
- 구체는 `<canvas>` + 직접 구현한 3D 투영/회전 로직, 나머지 배경은 SVG + CSS 3D
- 접근성: `prefers-reduced-motion`(모션 최소화) 환경에서는 애니메이션을 줄이고 콘텐츠를 즉시 노출
- 안전장치: 진입 애니메이션이 어떤 이유로 재생되지 않아도 본문 텍스트는 항상 보이도록 처리
- 다운로드용 단일 파일은 폰트·스크립트를 모두 내장 (오프라인 실행 가능)

## 7. 데이터 추가 방법

`리서치 아카이브.dc.html` 안의 `DATA` 배열에 항목을 추가하면 검색·분류·목록에 자동 반영됩니다.

```js
{
  title: "리서치 제목",
  file:  "ai-infra/파일명.html",   // 실제 리서치 파일 경로
  date:  "2026-06",
  category: "AI 인프라",            // 또는 "Physical AI"
  type:  "산업 개관",              // 산업 개관 / 종목 심화 / 투자 지도
  series: "AI 밸류체인",           // 시리즈가 없으면 null
  desc:  "한 줄 설명",
  tags:  ["전력", "GPU", "데이터센터"]   // 검색에 쓰이는 키워드
}
```

> 리서치 카드의 링크는 실제 리서치 파일들이 같은 폴더 구조(`ai-infra/`, `physical-ai/`)에 함께 있을 때 연결됩니다.

## 8. 손쉽게 조절 가능한 값

원할 때 알려주시면 바로 바꿔드릴 수 있는 항목들입니다.

- 구체 회전 속도 / 노드 개수(더 빽빽하게) / 색 흐름 속도
- 정지 후 구체가 남는 정도(더 사라지게 / 더 선명하게)
- 오프닝 길이와 타이밍
- 기본 진입 시안(네트워크 / 구체 / 레이어)
