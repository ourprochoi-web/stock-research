## 디자인 시스템

### 공통 요소
- **폰트**: Pretendard (CDN: cdn.jsdelivr.net) 단독 사용
- **파일 형식**: 단일 HTML, CSS/JS 인라인 (빌드 불필요)
- **Home 버튼**: 좌상단 고정 pill (`href="../"`, backdrop-filter blur)
- **Progress bar**: 스크롤 연동 상단 고정 바
- **Sticky nav**: pill 버튼형 섹션 네비게이션
- **Scroll reveal**: IntersectionObserver 기반 `.floor` 카드 fade-in
- **Count-up**: 히어로 통계 숫자 애니메이션
- **면책 조항**: 모든 파일 하단에 투자 비권유 고지 포함

### CSS 컴포넌트 패턴
| 클래스 | 용도 |
|--------|------|
| `.floor` | 메인 콘텐츠 카드 (border-top + scroll reveal) |
| `.zone-head` | 섹션 헤더 (큰 워터마크 번호 + 제목) |
| `.facts` | 3열 통계 그리드 |
| `.callout` | 컬러 헤더 강조 박스 |
| `.view` | 투자자 인사이트 패널 |
| `.analogy` | 인용/비유 박스 |
| `.gloss` | 아코디언 용어사전 (`<details>/<summary>`) |
| `.risk` | 리스크 카드 (심각도 색상 코딩) |
| `.tl` | 타임라인 |
| `.series-nav` | 시리즈 prev/next 네비게이션 |

### 파트별 고유 디자인
각 파일은 고유 악센트 컬러와 히어로 스타일을 가짐:
- **기업 분석**: 기업 정체성 반영 (한화=Gold, 현대로템=Military Blue 등)
- **심화 테마**: 테마 성격 반영 (엔진=불꽃, 우주=별, 해양=파도 등)
- 히어로 배경에 캔버스 애니메이션 또는 SVG 데코 적용
